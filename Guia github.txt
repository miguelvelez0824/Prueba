Crear un repositorio publico
Se conecta con git en la terminal de VSC usando los comandos de git y el link que proporciona github
    Comandos git:
              git init
              git status
              git add nombrearchivo
              git add .             (agrega todo lo que esta en rojo)
              git commit -m "mensaje"
              git push              (subir los archivos a github)
